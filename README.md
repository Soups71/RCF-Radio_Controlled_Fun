# RCF-Radio_Controlled_Fun

## Fun project to take my mind off my school stuff

Simple robot control firmware that I just spent an hour on.

Two firmware files:
    reciever.ino
    transmitter.ino

Materials:
    Parallax Joystick
    2 Arduino Uno's
    Robot Chasis
    2 continuous rotation servos
    2 nRF24L01 radio transmitter/recievers
